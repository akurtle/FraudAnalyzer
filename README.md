# Fraud Pattern Analyzer

Full-stack fraud pattern analysis project with:

- Python batch ingestion and anomaly detection
- PostgreSQL or Supabase-compatible persistence
- Simple frontend for uploading transaction data and reviewing flagged patterns
- PyTest coverage for integrity, partition isolation, retries, and regression scenarios

Project structure:

- `backend/` FastAPI API, fraud analysis pipeline, tests
- `frontend/` simple static frontend
- `docs/` architecture and design notes
